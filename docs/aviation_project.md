# Aviation Network Resilience Engine

## 🎯 Business Scenario
Airline hubs suffer from cascading delays, leading to massive rebooking costs. This project automates the identification of high-risk flights, allowing operational teams to intervene before delays spiral.

## 🛠 Tech Stack
- **Language:** Python
- **Database:** MongoDB
- **Logic:** Operational Research (Friction Index Formula)

## 💡 Key Contributions
- Designed an automated "Friction Index" that prioritizes flights based on delay and connection impact.
- Developed a synthetic data generator that mirrors real-world airport logistics.
- Built a proactive alerting system that flags high-friction scenarios for immediate intervention.
